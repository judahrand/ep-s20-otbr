Advanced

System management options. Use with care — some actions are irreversible without a backup.

Restart

Gracefully restart the border router. The web UI will become temporarily unavailable while the device reboots. All Thread network settings are preserved.

NVS Backup & Restore

Download a JSON snapshot of all NVS key-value entries (including the OpenThread dataset) or restore a previously saved snapshot.

Backup
Restore

Select a .json backup file produced by this device. Existing keys will be overwritten; keys not present in the backup are left untouched.
Restoring does not restart the device automatically: restart manually for changes to take effect.

Factory Reset

Erase all NVS settings, including the Thread dataset and any stored credentials. The device will restart after the erase.
This cannot be undone — download a backup first if you want to restore your settings later.

Type RESET in the box below and click "Confirm Reset" to proceed.
