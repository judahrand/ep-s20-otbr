OTA Update

Trigger a combined firmware update from a remote image URL or upload the images directly from your computer.

Remote Update

Enter the HTTP or HTTPS URL of an host and RCP OTA image produced for this border router.
After a successful download, the device will restart automatically and the web UI may become unavailable for a short time.

The URL must begin with http:// or https://.
Local Update

Select one or more firmware images to upload. At least one file must be selected.
Each field is optional — only the files you choose will be uploaded, in order: app, RCP, web.
After all uploads succeed the device will restart automatically and the web UI may become unavailable for a short time.

Host (ESP32) application firmware binary.
RCP OTA image in combined container format (no host firmware entry).
SPIFFS image for the web_storage partition (must be exactly 200 KB).
